# calculator_app

Expo Router app for a responsive calculator UI.

## What it does

- Evaluates expressions with `mathjs`.
- Supports portrait and landscape layouts.
- Uses safe area insets so the keypad and display stay readable on modern devices.
- Includes clear, all-clear, decimal, and operator handling.

## Run It

```bash
npm install
npm start
```

## Main Files

- `src/app/index.tsx`: calculator screen and interaction logic.
